F Major: F-G-A-Bb-C-D-E-F

Key Signatures: D Major: 2 sharps, Bb Major: 2 flats, A Major: 3 sharps, Eb Major: 3 flats

3. Intervals

Interval Identification: C to E: Major 3rd, F to A: Major 3rd, G to D: Perfect 5th, B to F: Diminished 5th

Building Intervals: C + Major 3rd = E, G + Perfect 5th = D, A + Minor 3rd = C, F + Perfect 4th = Bb

4. Rhythm and Time Signatures

Note Values in 4/4: Whole note = 4 beats, Half note = 2 beats, Quarter note = 1 beat, Eighth note = 1/2 beat

Complete the Measures: 4/4: add quarter note, 3/4: add quarter note, 2/4: add quarter note

5. Chords and Harmony

Triads: C Major: C-E-G, F Major: F-A-C, G Major: G-B-D, D minor: D-F-A

Chord Progressions in C Major: I = C Major, IV = F Major, V = G Major

6. Terms and Symbols

Dynamics: ff = fortissimo (very loud), mp = mezzo piano (moderately soft), crescendo = gradually getting louder, pp = pianissimo (very soft)

Tempo: Allegro = fast, Andante = walking pace, Presto = very fast, Largo = very slow
